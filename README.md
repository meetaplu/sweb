# Gem Tokens - Precious Gemstones on Blockchain

A premium next-gen landing page for tokenized precious gemstones. Own certified colored diamonds, rubies, emeralds & sapphires through blockchain-backed tokens.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3002
```

## ✨ Features

✅ Tokenized precious gemstones (diamonds, rubies, emeralds, sapphires)  
✅ 20% presale discount  
✅ Physical redemption - redeem tokens for actual gemstones  
✅ "How Redemption Works" section  
✅ "Available Gem Lots" showcase  
✅ Company information & contact details  
✅ Whitepaper placeholder (Coming Soon)  
✅ Smooth scrolling with Lenis  
✅ Scroll-triggered animations with GSAP  
✅ Fully responsive  
✅ TypeScript support  
✅ Premium luxury design

## 🛠️ Tech Stack

- **Next.js 15** - React framework
- **React 19** - UI library
- **TypeScript** - Type safety
- **Tailwind CSS** - Premium styling
- **GSAP + ScrollTrigger** - Scroll animations
- **Lenis** - Smooth scrolling
- **SplitType** - Text animations
